## Here are the available commands:

### /help - Show this help message.
### /register - Register for the service.
### /login - Log in to your account.
### /logout - Log out of your account.
### /recommend - Get personalized recommendations.
### /profile - View your profile information.
### /update - Update your profile information.
### /delete - Delete your account.


## Libraries Used 

### ```pip install  PytelegramBotAPI```
### ```pip install  Sqlite3```
### ```pip install  dotenv```


## How to use 

### 1. Use /Start
### 2. Register using /register 
### 3. answer the questioner/complete registration
### 4. use /recommend to get recommended something
### 5. use /update to update info about you and use /info to check info about yourself
