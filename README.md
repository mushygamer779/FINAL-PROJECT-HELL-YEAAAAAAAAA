# BOT RIZZLER ☆*:.｡.o(≧▽≦)o.｡.:*☆

Welcome to **BOT RIZZLER**! This is a Telegram bot designed to handle user registrations, profiles, and deliver personalized recommendations.

---

## 🚀 Features & Commands

Here is a quick breakdown of what the bot can do:

| Command | Description |
| :--- | :--- |
| `/start` | Launch the bot and get started. |
| `/help` | Show the help message with available commands. |
| `/register` | Start the registration questionnaire. |
| `/login` | Log in to your existing account. |
| `/logout` | Log out of your current session. |
| `/recommend` | Get personalized recommendations. |
| `/profile` | View your profile information. |
| `/info` | Check your current profile data. |
| `/update` | Update your profile information. |
| `/delete` | Permanently delete your account. |
